WHAT’S

TOFFY

MAP ?

Toffy map คือ ผู้ให้บริการข้อมูลด้านสื่อโฆษณานอกบ้าน (Out-of-Home Media : OOH) ในรูปแบบ เชิงพื้นที่ทั้งในรูปแบบ ฐานข้อมูล ระบบ Web Application เพื่อการสืบค้นและแสดงข้อมูลสรุป รวมถึง รายงานรูปเล่มพร้อมบทความเชิงวิเคราะห์ ด้วยรูปแบบการนำเสนอที่หลากหลาย เพื่อช่วยให้ท่านเข้าใจ สภาพการลงทุนในสื่อโฆษณานอกบ้านได้อย่างครบทุกมิติ และทันสมัยมากที่สุด

PERFECTLY BILLBOARD

INFORMATION

Toffy map ทำการรวบรวมข้อมูลป้ายโฆษณา (Billboard) หลากหลายรูปแบบ ไม่ว่าจะเป็น ป้ายขนาดใหญ่ ข้างทาง,ขนาดเล็กตามฟุตบาธ หรือจะเป็นป้ายยอดตึกต่างๆ ประกอบกับข้อมูลที่มีประโยชน์ เช่น ขนาด เจ้าของป้าย ราคาอ้างอิง รายละเอียดโฆษณาบนป้าย รวมถึงภาพประกอบ และ คะแนนคุณภาพของ ป้ายที่ประเมินจากปัจจัยหลายด้าน เพื่อให้ผู้ใช้งานสามารถนำข้อมูลไปต่อยอดวิเคราะห์การลงทุนได้อย่าง มีประโยชน์สูงสุด

APPLICATION

FOR

ANALYSIS

Toffy map นำเสนอข้อมูลป้ายโฆษณาในรูปแบบ Web Application สำหรับการค้นหาสืบค้นข้อมูล ที่มีประสิทธิภาพ รวมถึงระบบแสดงผลข้อมูลในภาพรวม (Operation Dashboard) ผู้ใช้งานสามารถ เลือกรูปแบบ Application ตามมาตรฐาน หรือให้เราพัฒนาเครื่องมือรูปแบบเฉพาะให้เหมาะกับองค์กร ของท่านเอง เพื่อช่วยให้ท่านสามารถบริหารจัดการ และติดตาม campaign ของท่านได้อย่าง มีประสิทธิภาพสูงสุด
